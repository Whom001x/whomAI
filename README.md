# 悬浮窗 AI 对话

一个 Windows 桌面悬浮 AI 对话小工具。

## 功能

- 默认固定在其他网页、文档、软件窗口上方
- 无系统标题栏和外边框，只保留软件自己的顶部控制栏
- 顶部“菜单”集中管理钉住、透明模式、短回答模式、清空聊天、最小化、设置、退出
- 取消固定后可以拖动顶部标题栏移动窗口
- 鼠标拉住窗口四周窄边可以调节大小
- 支持选择图片后发送给 AI 分析
- 支持在输入框按 Ctrl+V 从剪贴板粘贴图片
- 支持 Ctrl+Enter 发送
- 支持设置全局快捷键最小化/恢复窗口，默认 `Ctrl+Alt+A`
- 支持配置 API Key、模型、接口地址、透明度、系统提示词
- 普通透明度支持设置到 `0.01`
- 配置保存到同目录 `app_config.json`

## 运行

双击 `FloatingAIChat.exe`。

源码版可以双击 `run.bat`，或在当前目录运行：

```powershell
python floating_ai_chat.py
```

第一次运行后，点击右上角“设”填写 API Key。

全局快捷键可以在“菜单 -> 设置 -> 全局快捷键”里修改。点击输入框后直接按键盘组合即可录制，例如按住 Ctrl+Alt 再按 A。

发送图片时，点击输入框上方的“图片”选择文件，再输入问题并点击“发送”。也可以只选择图片直接发送。

如果剪贴板里有截图或图片，在输入框按 `Ctrl+V` 会自动选中这张图片；顶部菜单里也有“粘贴图片”。

底部“清图”只清除当前已选图片；清空聊天请打开顶部“菜单”选择“清空聊天”。

默认是短回答模式，AI 会尽量用 1-3 句回答。已有旧配置时，可以打开顶部“菜单”选择“短回答模式”。

打开顶部“菜单”选择“透明模式”后，窗口背景会挖空透明，只保留文字、输入框和必要按钮。

输入框颜色可以在“菜单 -> 设置 -> 输入框颜色”里选择，普通模式和透明模式都会一起生效。

设置里也可以点“屏幕取色”，然后点击屏幕任意位置，把该像素颜色填入输入框颜色。

支持 PNG、JPG、WebP、GIF，单张图片建议不超过 20MB。

也可以不在软件内保存 Key，改用环境变量：

```powershell
$env:OPENAI_API_KEY="你的 API Key"
python floating_ai_chat.py
```

## 默认接口

默认调用 OpenAI Responses API：

```text
https://api.openai.com/v1/responses
```

默认模型：

```text
gpt-5.5
```
